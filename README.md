# CareLens AI

AI-powered Clinical Decision Support & Medical Report Analysis Platform.

## Features

- OCR
- AI Report Analysis
- Clinical Summary
- Lab Value Detection
- Patient-Friendly Explanation
- RAG Knowledge Base
- Medical Chat
- Report History

## Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- FastAPI

AI:
- LangChain
- Google Gemini
- Tesseract OCR

Database:
- Supabase PostgreSQL

Vector Database:
- FAISS
